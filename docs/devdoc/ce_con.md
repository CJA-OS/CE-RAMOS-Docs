# 插件投稿
:::tip 注意：上传时请把插件命名成以下格式：
```
软件名_作者(你)_软件版本号_插件介绍.ce

如：
CE-基础维护包_1.2.1_CJA_CE-RAMOS官方推出的PE基础维护包.ce
```
请确保插件:未重复(可为新版)、路径使用相对路径或环境变量、无害，文件名不超过80个字符，可在CE-RAMOS内正常加载

上传插件后，会对你的插件进行评估，通过则加入插件市场
:::
